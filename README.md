# Gym Landing Page
Landing page moderna y responsiva para gimnasio, construida con React y Tailwind CSS.

## Tecnologias
- React
- Tailwind CSS
- Vite
- React Icons

## Features
- Diseño responsivo (mobile-first)
- Navegacion smooth scroll
- Formulario de contacto
- Secciones: Hero, About, Classes, Trainers, Contact
